```
yarn
```

## Procédure

### 1 - installation de NextJS
Si vous êtes déjà dans un dossier de travail
```bash
yarn create next-app --typescript .
```
sinon
```bash
yarn create next-app --typescript <NOM_DE_DOSSIER>
```
